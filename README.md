## 일랜시아 루트 계산기

![routegif](https://i.imgur.com/3nxjGrJ.gif)

### :joystick: 넥슨 고전게임 일랜시아의 직업포인트에 따른 스탯 계산기입니다.

### :construction: 설치 및 실행

```sh
npm i
npm start
# http://localhost:1234 에서 계산기 실행가능
```

### :rocket: 빌드

```sh
npm i
npm run build
```

### :hammer: 테스트

```sh
npm i
npm test
```

### 기술 스택

- React
- TypeScript
- bulma
- parceljs
- jest
