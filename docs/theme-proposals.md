# Theme Proposals — Route Calculator

간단 계획

- 현재 코드에 이미 적용된 시안(간단 요약) 기록
- 이미지(레트로 MMORPG HUD)를 참고해 후보 테마 5개 정리
- 각 테마에 색상 팔레트·3D 강도·적용 포인트·왜 어울리는지 짧게 적음
- 사용자가 선택하면 적용 절차(파일 목록 + 변경 포인트) 제공

---

## 1) 현재 적용된 테마 (요약)

- 이름(임의): Muted / Win98 hybrid (현재 작업 상태)
- 적용된 파일
  - `src/App.tsx`
    - Title 스타일 업데이트: 배경을 `#3e3e42`, 테두리 상/좌 `#6f6f73`, 하/우 `#222326`로 변경
    - 직업 버튼들: 회색 배경(`#c0c0c0`), 검정 텍스트, subtle bevel(상/좌 밝음, 하/우 어두움), 미세한 shadow, 패딩 축소(`px-2 py-1`), 모서리 `rounded-none`으로 각진 형태
  - `src/components/ElanBox.tsx`
    - (주의) `ElanBox`의 내부 경계와 그림자 패러미터는 원본에 남아있음 — 필요하면 여기도 메탈/패널 톤으로 통일 가능
- 시각적 요점
  - 전반적으로 DOS/Win98 방향과 현대적 요소(미세한 그림자/포커스 링)를 혼합한 상태
  - 비교적 차분한 회색 계열 유지, 버튼은 플랫에 약간의 베벨만 남김

---

## 2) 이미지 기반 후보 테마 (요청에 따른 5개)

아래 후보는 첨부하신 게임 UI 스크린샷(레트로 MMORPG HUD)을 참고해 구성했습니다.

1. Classic MMO HUD (원본 복각 스타일)

- 팔레트: 배경 `#1a1a1a`, 패널 `#161617`, 타이틀 `#222326`, 텍스트 `#e6e6e6`, 액센트/게이지 `#00b8d9`
- 3D 강도: 뚜렷한 1–2px 베벨 (strong)
- 적용 포인트: Title/ElanBox 테두리, 게이지(HP/MP) 색, 우측 아이콘 프레임
- 이유: 스크린샷과 스타일 매칭이 가장 높음 (원본 복각 목적)

2. Slate + Cyan Gauge (모던 레트로 믹스)

- 팔레트: 배경 `#202328`, 패널 `#2b2f33`, 타이틀 `#31343a`, 텍스트 `#dbe6ea`, 액센트 `#00c0b8`
- 3D 강도: 얕은 베벨 + soft drop shadow (subtle-modern)
- 적용 포인트: Title에 크롬 라인, 패널 내부 구획에 얕은 그림자, 게이지에 시안 포인트
- 이유: 원본 무게감을 유지하면서 현대적 가독성을 추가

3. Industrial Steel (무광 금속)

- 팔레트: 패널 `#2f3236`, 타이틀 `#3b3f43`, 텍스트 `#f1f3f4`, 액센트 `#9fcbe0`
- 3D 강도: 미세 베벨 + 1px 하이라이트 라인 (metallic)
- 적용 포인트: ElanBox 외곽선과 Title 라인, 버튼 테두리에 금속성 하이라이트
- 이유: 무게감과 중후함을 강조하고 싶을 때

4. Retro Pixel Frame (픽셀/클래식)

- 팔레트: 패널 `#cfcfcf`(연회색), 텍스트 `#000000`, 액센트 `#00a4ff`
- 3D 강도: 강한 베벨(각진 2px 테두리) — 픽셀 스타일 (very retro)
- 적용 포인트: 모든 프레임/버튼/아이콘 프레임을 각지게 처리
- 이유: 도스/윈98 시대의 블록 UI를 강하게 재현

5. Warm CRT (따뜻한 복고 + 소프트 글로우)

- 팔레트: 패널 `#292622`, 타이틀 `#332f2b`, 텍스트 `#efe8dd`, 액센트 `#6fe0c6`
- 3D 강도: 소프트 베벨 + 아주 약한 내부 글로우 (retro-soft)
- 적용 포인트: 타이틀 로고 주변 약한 글로우, 게이지/선택 버튼에 색 번짐
- 이유: 차갑지 않은 복고미를 원할 때 유용

---

## 3) 적용 가이드 (선택 후 빠르게 적용하기)

- 1단계: `docs/theme-proposals.md`에서 테마 번호 선택
- 2단계: 선택된 테마의 색상 팔레트를 `src/App.tsx` (Title)과 `src/components/ElanBox.tsx` (프레임/테두리)에 반영
- 3단계: 직업 버튼(모든 className)에서 배경/테두리/텍스트 색상만 조정 (현재 구조 유지)
- 4단계: (선택사항) 게이지/테이블/아이콘 색상도 팔레트에 맞춰 조정

파일 변경 포인트(예시)

- `src/App.tsx` — Title 클래스, 버튼 className 블록(1차/2차/모험가/상인)
- `src/components/ElanBox.tsx` — 바깥 배경/내부 테두리/boxShadow
- (옵션) `src/components/ElanButton.tsx` — 상단 버튼 테마 통일
